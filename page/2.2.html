2.2 Операторы break, continue

Оператор continue

Оператор continue начинает следующий проход цикла, минуя оставшееся тело цикла (for или while)

for i it 'hello world':
    if i == 'o':
        continue
    print(i * 2, end='')

Оператор break

Оператор break досрочно прерывает цикл.

for i it 'hello world':
    if i == 'o':
        break
    print(i * 2, end='')
